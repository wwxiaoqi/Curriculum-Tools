# Curriculum Tools
该项目用于把「年级班级安排表」变成「班级课程表」

* Windows 10
* python >= 3.6

------------------------------------

![Curriculum Tools](https://wwxiaoqi-1253423189.cos.ap-guangzhou.myqcloud.com/Github/Curriculum_Tools/Curriculum_Tools.png)


使用方法：
-------------
```
git clone https://github.com/wwxiaoqi/Curriculum-Tools
cd Curriculum-Tools
python main.py
```
接着查看你的文件夹里面的：
* *.pdf
* *.png
